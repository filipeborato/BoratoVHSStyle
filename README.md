# BoratoVHSStyle

A retro VHS-style UI theme library for JUCE plugins.

## Features

- **VHS Aesthetic**: Neon colors, scanlines, noise, and grid backgrounds
- **Custom Controls**: VHS-styled knobs, sliders, buttons, and meters
- **Audio Reactive**: Optional shimmer effects that respond to audio levels
- **Easy Integration**: Simple API to apply theme to your plugin

## Building

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## Usage

See `examples/DemoPlugin` for a complete example.
